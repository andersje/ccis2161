• arithmatic: BASH assumes everything is a string variable, unless you tell it differently
    • jeremy@jeremy-desktop:~/Desktop$ X=3+4
jeremy@jeremy-desktop:~/Desktop$ echo $X
3+4
  • use let to get around that
    • jeremy@jeremy-desktop:~/Desktop$ let X=3+4
jeremy@jeremy-desktop:~/Desktop$ echo $X
7
